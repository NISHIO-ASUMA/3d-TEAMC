[ ゲームの世界観 ]-------------------------------------------------

オタクの青年A君がオタクをバカにしてくる人間を侍に扮して制裁していく
3Dアクションでハックアンドスラッシュスタイルのゲーム

[ インゲーム関係 ]--------------------------------------------------
・敵を様々な武器を用いてどんどん吹っ飛ばすアクション
・落ちているものは拾って武器として扱え,まとめて制裁 / 石などは投擲可能
・フィーバーモード時は攻撃力と移動量が大幅に増加 高速アクションが可能に
・100秒の中でどれだけ倒したかのスコアを競います

[ 操作方法 ]--------------------------------------

画面遷移 → Enterキー / マウスの左クリック/ パッドのAボタン 
カメラは基本プレイヤー追従

[ キーマウ ]
移動 → W/A/S/D
攻撃 → マウスの左クリック(トリガー)
SP攻撃 → SPゲージが溜まっている状態で Qキー

ジャンプ → SPACEキー
カメラ操作 → マウス移動(前後左右)
回避操作 → マウスの右クリック (トリガー)
ポーズメニュー → Pキー

武器を拾う → Eキー 
拾ったもののストック → Fキー
武器を投げる(石とか投げられそうなもの) → マウスの左クリック(トリガー)
武器の合成 / 終了 → TABキー
クラフト決定 → マウスの左クリック(トリガー)

[ パッド ]
移動 → Lスティック
攻撃 → Xボタン
SP攻撃 → SPゲージが溜まっている状態で Lスティック押し込み

ジャンプ → Aボタン
カメラ操作 → Rスティック
回避操作 → Bボタン
ポーズメニュー → STARTボタン

武器を拾う → LB 
拾ったもののストック → RB
武器を投げる(石とか投げられそうなもの) → Xボタン(攻撃ボタンと同じ)
武器の合成 / 終了 → Yボタン
クラフト決定 → Aボタン
