[ ゲームの世界観 ]-------------------------------------------------

オタクの青年A君がオタクをバカにしてくる人間を侍に扮して制裁していく
3Dアクションでハックアンドスラッシュスタイルのゲーム

[ インゲーム関係 ]--------------------------------------------------
・敵を様々な武器を用いてどんどん吹っ飛ばすアクション
・落ちているものは拾って武器として扱え,まとめて制裁 / 投擲可能
・フィーバーモード時は攻撃力と移動量が大幅に増加 高速アクションが可能に
・100秒の中でどれだけ倒したかのスコアを競います

[ 操作方法 ]--------------------------------------

画面遷移 → Enterキー / パッドのAボタン
カメラは基本プレイヤー追従

[ キーマウ ]
移動 → W/A/S/D
攻撃 → マウスの左クリック(トリガー) / Enterキー
ジャンプ → SPACEキー
カメラ操作 → マウス移動(前後左右)
武器を拾う → Fキー 
武器を投げる(石とか投げられそうなもの) → マウスの左クリック(トリガー) / Enterキー

[ パッド ]
移動 → Lスティック
攻撃 → Xボタン
ジャンプ → Aボタン
カメラ操作 → Rスティック
武器を拾う → LB 
武器を投げる(石とか投げられそうなもの) → RB

[ 制作中の要素 ]----------------------------------
1.武器の合成システム
2.演出,パーティクルなどの改良
