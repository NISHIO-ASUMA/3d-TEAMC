[ 操作方法 ]--------------------------------------

画面遷移 → Enterキー / パッドのAボタン
カメラは基本プレイヤー追従

[ キーマウ ]
移動 → W/A/S/D
攻撃 → マウスの左クリック(トリガー) / Enterキー
ジャンプ → SPACEキー
カメラ操作 → マウス移動(前後左右)
武器を拾う → Fキー 
武器を投げる(石とか投げられそうなもの) → マウスの左クリック(トリガー) / Enterキー

[ パッド ]
移動 → Lスティック
攻撃 → Xボタン
ジャンプ → Aボタン
カメラ操作 → Rスティック
武器を拾う → LB 
武器を投げる(石とか投げられそうなもの) → RB
